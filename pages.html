How to Publish Web Content: How to Publish a single Page for special content

What is a Page?

A Page is used for static content, like the About WILL page. Pages aren't part of a series of posts displayed by date like on the News page. Use the WILL Pages section to create a singular stand-alone web page, like the _About_ page.

How to create a Page

  1. Log into ExpressionEngine and go to create an entry like any other post.
  2. In the dropdown list of sections to post to, select WILL Pages.

The entry form for WILL Pages is almost the same as for any other post, with one exception. Near the top of the entry form you'll see a _Pages URI_ field and a _Template_ dropdown list:

In the _Pages URI_ field, you can assign a custom url to the page you're creating. Enter only the part of the url after _will.illinois.edu_. In the case of the _About_ page, we want to final url to be _will.illinois.edu/about_, so we just need to enter _/about_ in the _Pages URI_ field.
